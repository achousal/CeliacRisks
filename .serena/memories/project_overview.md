# CeD-ML Project Overview

Primary documentation: [analysis/CLAUDE.md](../../analysis/CLAUDE.md)

**Quick facts:**
- ML pipeline for incident Celiac Disease risk prediction from proteomics
- Dataset: 43,960 samples, 2,920 proteins, 0.34% prevalence
- Models: RF, XGBoost, LinSVM_cal, LR_EN (nested CV, Brier optimization)
- Stats: 15,109 lines, 753 tests, 82% coverage
