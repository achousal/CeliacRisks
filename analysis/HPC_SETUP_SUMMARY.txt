QUICK START COMMANDS
================================================================================

On HPC:
-------
1. Clone:     git clone git@github.com:achousal/CeliacRisks.git
2. Setup:     cd CeliacRisks/analysis && bash scripts/hpc_setup.sh
3. Activate:  source venv/bin/activate
4. Submit:    bsub < CeD_production.lsf

On Local (after jobs complete):
-------------------------------
5. Download:  rsync -avz user@hpc:/path/to/results/ ./results/
6. Analyze:   Rscript compare_models_faith.R --results_root results/
